2.1. Power Calculator

To start Power Calculator from the Lattice Radiant software Tools menu or toolbar: 1. Open a project or create a new one. 2. Choose Tools > Power Calculator. Power Calculator main window opens in Estimation Mode or Calculation Mode, depending on the design stage, and displays the Power Summary page as shown in Figure 2.1.
