Start Page

This section describes the major sections of the Start Page in the Radiant software.

The Start Page contains the following three major sections:
  • Project: This section allows you to create a new project; open an existing Project, and open an example.
  • Information Center: This section has links to Getting Started, Tutorials, User Guides, and Support Center.
  • Recent Project List: Provides a quick way to load a recent project youâ€™ve been working on.

The Start Page appears in the View area by default when the Radiant software is first launched, and can be opened from the View tab on the menu. The Start Page can be closed, opened, detached, and attached using the Attach button.
