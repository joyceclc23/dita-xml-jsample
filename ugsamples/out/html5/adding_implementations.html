Adding Implementations

This section describes how to add implementations.

To add a new implementation to an existing project:, follow these steps:

  1. Right-click the project name in the File List project view.
  2. Select Add > New Implementation.
  3. In the New Implementation dialog box, set the implementation name, directory, and default strategy.
  4. To browse for the source files or use a source from an existing implementation, click on Add Source.
